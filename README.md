System-V banner clone with russian cyrillyc support
===================================================
Writed on Golang

## Run
```bash
make run MESSAGE
```

## Build
```bash
make
```

## Instal (to /usr/local)
```bash
sudo make install
```

## Run simple clock demo
```bash
./clock.sh
```

## Uninstal (from /usr/local)
```bash
sudo make uninstall
```
